# IMAD-ASSIGNMENT-1
## PURPOSE OF THE APP
The app is designed to suggest meaning suggestions to cora throughout the day. The user enters their name and time of day, and the app will provide a suitable recomendation based on the input. The purpose of the app is to encourage communication in a simple and practica way, providing guidance through small social interactions for various parts of the day. The app needed to be clear and understandable with clear labels as where to enter information. video link https://youtu.be/_yfITsETb8Q

## GITHUB
GitHub was used to store the project code online, track changes, and manage versions of the app during development.
## GITHUB ACTIONS
GitHub Actions was used to automate the build process, helping to check that the app still compiled correctly whenever code was pushed. By automating this process, the project became more robust and maintainable. This also made testin much easier and improved consistincy.This helps supports good software development practice because it helps identify issues early and keeps the application in a working state during development.
